3. 활용 예시 — RPG 보드게임 시뮬레이션

Trigger + Lorebook + Variables 연동
📜 Lorebook 설정
키워드: "상점", "구매"
내용: 상점 시스템 규칙 설명 (가격표, 거래 규칙)
조건: getVar("location") == "town"
⚡ Trigger: on_user_message — 패턴 "구매|살|샀"
local item = matchRegex(msg, "구매%s+(.+)")
local gold = getVar("player.gold")
local price = prices[item] or 10

if gold >= price then
  setVar("player.gold", gold - price)
  setVar("inventory."..item, true)
  sendMessage("✅ "..item.." 구매! (남은 골드: "..(gold-price)..")", "system")
  playEffect("coin_sound")
else
  sendMessage("❌ 골드가 부족합니다! (필요: "..price..", 보유: "..gold..")", "system")
  playEffect("error")
end
📊 현재 변수 상태
player.hp = 78 | player.gold = 45
location = "town" | quest.main = "dragon_slain"
 inventory.sword = true | flags.met_king = true
사용자에게 보이는 화면
📍 마을 🌙 저녁 💰 골드: 45 ❤️ HP: 78
상점 주인이 당신을 반기며 웃습니다. 진열대에는 다양한 물약과 무기가 놓여 있습니다.
나
"포션 구매"
✅ 포션 구매! (남은 골드: 35)
상
"좋은 선택이에요, 모험자님. 이 포션이라면 상처도 금방 치료될 겁니다. 또 필요한 건 있으신가요?"
📖 Lorebook 활성: 상점 시스템 규칙 (320토큰)
