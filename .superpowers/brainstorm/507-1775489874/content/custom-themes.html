1. 비주얼 노벨 스타일

테마: 비주얼 노벨 — 캐릭터 초상화 + 대화창
🧝
⚔️
아리아
"이 문양... 고대 엘프어로 쓰여 있어요. 조심하세요, 여기 함정이 있을 수도 있어요."
▼ 클릭하여 계속
템플릿 구조: 전체 화면 배경 + 캐릭터 포즈/표정 + 하단 대화창 오버레이. 메시지 단위가 아닌 "장면 단위" 렌더링.

2. 라이트 노벨 / 소설책 스타일

테마: 소설책 — 종이 질감 + 활자체
CHAPTER III
깊은 숲 속의 유적
━━━━━━━━━━━━━━━━━━━━━

달빛이 희미하게 비추는 숲속, 세 사람은 고대 유적의 입구 앞에 섰다. 차가운 바람이 불어올 때마다 유적의 벽에 새겨진 문양이 희미하게 빛났다.

"이 문양... 고대 엘프어로 쓰여 있어요." 아리아가 지팡이를 들어 벽의 문양을 비추며 말했다. "조심하세요, 여기 함정이 있을 수도 있어요."

카이는 검을 뽑아들었다. "함정? 좋아, 내가 먼저 들어가지." 그는 주저 없이 유적 안으로 발을 들였다.

나는 카이를 말리려 했다. "잠깐, 먼저 문양을 더 자세히 살펴보자." 아리아의 어깨 너머로 벽의 문양을 들여다보았다.

─ ─ ─ ─ ─
템플릿 구조: 전체 텍스트를 하나의 연속된 소설 본문으로 렌더링. 대화는 인라인 따옴표, 행동은 서술문, 캐릭터별 색상 구분. 페이지네이션 지원.

3. TRPG 세션 로그 스타일

테마: TRPG — 다이스 + 세션 로그
⚔️ Session #3 — 어두운 숲의 모험 Turn 12
[GM] 달빛이 희미하게 비추는 숲속, 세 사람은 고대 유적의 입구 앞에 섰다.
아리아 [지혜 체크: 성공 17]
"이 문양... 고대 엘프어로 쓰여 있어요. 조심하세요, 여기 함정이 있을 수도 있어요."
카이 [용기 체크: 성공 19]
"함정? 좋아, 내가 먼저 들어가지."
→ 카이는 검을 뽑아 유적 안으로 진입했다.
플레이어 [지혜 체크: 실패 8]
"잠깐, 먼저 문양을 더 자세히 살펴보자."
→ 아리아에게 다가가 벽의 문양을 함께 살펴본다.
⚙ 시스템: 장면 업데이트 — 유적 내부 탐험 시작. 다이스 롤 대기 중...
템플릿 구조: 턴 기반 로그. 다이스 롤 결과 표시, GM 서술, 캐릭터 행동 + 체크 결과. 시스템 메시지가 게임 로그처럼 표시.

4. 하이브리드 메신저 + 서술

달빛이 희미하게 비추는 숲속, 고대 유적 앞 — 밤
