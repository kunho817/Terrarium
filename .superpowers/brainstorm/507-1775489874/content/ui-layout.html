라우팅 구조

/ → 채팅 목록 (홈)
/chat/[id] → 채팅 화면
/chat/[id]/info → 채팅/캐릭터 정보 패널 (슬라이드인)
/characters → 캐릭터 갤러리
/characters/new → 캐릭터 생성/편집
/settings → 설정
/settings/providers → API 프로바이더 설정
/settings/plugins → 플러그인 관리
