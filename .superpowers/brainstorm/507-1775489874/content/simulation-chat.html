B. 커스텀 HTML/CSS 테마 적용 예시

사용자가 HTML/CSS로 직접 채팅 뷰를 커스터마이징 가능

동일한 채팅 — 사용자 커스텀 테마 적용
━━━ 깊은 숲 속, 고대의 유적 앞 ┃ 밤 ━━━

달빛이 희미하게 비추는 숲속, 세 사람은 고대 유적의 입구 앞에 섰다. 차가운 바람이 불어올 때마다 유적의 벽에 새겨진 문양이 희미하게 빛났다.

[아리아] "이 문양... 고대 엘프어로 쓰여 있어요. 조심하세요, 여기 함정이 있을 수도 있어요."

[카이] "함정? 좋아, 내가 먼저 들어가지."

[나] "잠깐, 먼저 문양을 더 자세히 살펴보자."

이 뷰는 사용자가 작성한 HTML/CSS 템플릿으로 렌더링됩니다.
<div class="message" data-char="아리아">{{content}}</div> 같은 템플릿 변수 사용

UI 커스터마이징 시스템 구조

// 사용자 커스텀 테마 예시
theme.html:
<div class="msg" data-type="{{type}}">
<span class="name" style="color:{{char.color}}">{{char.name}}</span>
<div class="content">{{content}}</div>
{{#if action}}<i class="action">{{action}}</i>{{/if}}
</div>

theme.css:
.msg[data-type="narrator"] { border-left: 3px solid #666; font-style: italic; }
.name { font-weight: bold; }
.action { color: #888; }
