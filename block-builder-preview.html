Key ComfyUI Features to Consider

🔍 Search-Based Node Palette

Double-click canvas → search popup → select block. Much faster than scrolling a sidebar for large block libraries.

High Priority

🔍 Zoom & Pan Navigation

Scroll to zoom, drag empty canvas to pan. Essential for graphs with many nodes. Current fixed 500px canvas feels cramped.

High Priority

🗺️ Minimap

Bird's eye view showing all nodes. Click to jump to that area. Critical for navigating large graphs.

High Priority

🖥️ Full-Screen Canvas

Remove fixed 3-column layout. Canvas takes full viewport, panels float or dock on demand. Better use of screen space.

Medium Priority

📁 Collapsible Nodes

Click header to collapse node to just the title bar. Reduces visual clutter when working on complex graphs.

Medium Priority

📋 Context Menu

Right-click on canvas for quick actions: Add Node, Paste, Select All. Right-click on node: Delete, Duplicate, Collapse.

Low Priority
