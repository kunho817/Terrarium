How It Works

Left Panel: Block palette with draggable components. Drag any block onto the canvas.
Center Canvas: Visual workspace where you arrange and connect blocks. Click and drag from output ports (right side) to input ports (left side) to create connections.
Right Panel: Live preview shows the generated prompt in real-time. Toggle panel lets you test different toggle states.
